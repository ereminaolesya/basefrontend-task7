## Live Demo

https://ereminaolesya.github.io/basefrontend-task7/

## Описание

Реализация HTML+CSS проект адаптивной сетки карточек товаров с использованием Flexbox.
Задача взята из учебного практического курса.

## Требования к задаче
Технические ограничения:
Запрещено использовать display: grid и связанные с Grid свойства. Задание направлено на изучение исключительно Flexbox-позиционирования.
Запрещено использовать фиксированные, абсолютные и относительные позиционирования

Основные требования:
В проекте должно быть использование display: flex как минимум 4 раза
Карточки должны смещаться на новый ряд при недостатке места в текущем
Необходимо соблюсти семантику тегов

Визуальные особенности:
Обратите внимание, что изображения телевизоров имеют разные разрешения
